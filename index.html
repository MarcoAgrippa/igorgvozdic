About Me

Hi, my name is Igor and I’m a QA automation engineer. My passion for programming happened by accident while I was trying to build a web page for fun. After that, I got interested in Java and especially Android but then I realized that I have to go back to CS basics so I read a few books and watched a couple of courses about programming fundamentals. From then on I tried to write my first fully functional Java EE and Android apps. The knowledge that I gained during this process got me my first IT job and now I’m perfecting my skills as a QA Engineer. I’m always on the lookout to learn something new, so at the moment I’m learning Spring Boot and Selenium. I speak English and Italian and trying to learn German.
